========================================
    村庄模拟塔防游戏 - 策划案 v3.0
    当前实现: 无构建步骤的 HTML/CSS/JavaScript 模块
========================================

【一、游戏概述】
--------------------------------
类型：村庄模拟 + 塔防（浏览器游戏）
视角：俯视网格地图 80×50 格（3200×2000 像素），支持相机平移+滚轮缩放
美术：简约几何体（圆形建筑、椭圆居民）
世界观：部落冲突风格
核心循环：白天生产建设 → 夜晚塔防战斗

【二、核心循环】
--------------------------------
白天：
  居民劳作 → 产出资源 → 搬运到对应仓库 → 消耗资源下蓝图 → 工程师建造 → 扩张
夜晚（22:00-6:00）：
  怪物从入夜视野边界外的迷雾前线刷新 → 寻路冲向大本营 → 遇建筑攻击摧毁 → 箭塔自动防御
  居民回家消失，天亮出现

【三、建筑系统】
--------------------------------
所有建筑统一网格放置。升级大本营解锁新建筑。
建筑等级上限 = 大本营等级。
建筑有两个面积：虚线网格占地 + 实心圆形碰撞体（1x1=14px半径, 2x2=34px半径）

大本营升级消耗：
  Lv.1→2: 20木 | Lv.2→3: 50木 | Lv.3→4: 200木+50石

解锁等级：
  一本(Lv.1): 农场、伐木场、箭塔、道路
  二本(Lv.2): 住房、木材仓库、食物仓库
  三本(Lv.3): 采石场、石材仓库、城墙
  四本(Lv.4): 铁矿、炼铁机、铁矿仓库、铁锭仓库

建造时间(buildTime): 一解2s、二解5s、三解8s、四解11s（每级+3s）

建筑定义(BLD_DEFS):
  农场:      2x2, 5木, HP100, 2人, 食物, 8s, unlock:1
  伐木场:    2x2, 5木, HP100, 2人, 木材, 6s, unlock:1
  箭塔:      1x1, 20木10石, HP120, 射程150, 伤害15, CD0.8s, unlock:1
  住房:      2x2, 10木5石, HP200, popBonus+4, 升级+2, unlock:2
  木材仓库:  2x2, 10木5石, HP150, 容量50, unlock:2
  食物仓库:  2x2, 10木5石, HP150, 容量50, unlock:2
  采石场:    2x2, 10木5石, HP120, 3人, 石材, 10s, unlock:3
  石材仓库:  2x2, 10木5石, HP150, 容量50, unlock:3
  城墙:      1x1, 5石, HP250, unlock:3
  铁矿:      2x2, 15木10石, HP150, 4人, 铁矿, 12s, unlock:4
  炼铁机:    2x2, 20木15石, HP120, 2人, 铁锭, 8s, unlock:4
  铁矿仓库:  2x2, 10木10石, HP150, 容量50, unlock:4
  铁锭仓库:  2x2, 10木10石, HP150, 容量50, unlock:4
  大本营:    2x2, 免费(初始), HP500, 可住人(4+4*等级), 可招募居民(5食物/人,10s)

建筑交互（左键点击 -> 弹出菜单）：
  - 升级：消耗资源，升级时间=buildTime。住房+2人口，大本营+4人口+解锁新建筑
  - 增减工人：不超上限，满/空时灰色disabled。点击不关菜单
  - 拆除：删除建筑，工人变无业。大本营不可拆
  - 移动：半透明虚影，绿/红网格。大本营不可移动
  - 蓝图：透明50%，不显示工人操作，不可分配工人

【四、蓝图建造系统】
--------------------------------
放置建筑 -> 创建蓝图（不检查资源，50%透明，HP=1）
-> 工程师从仓库取材料（每次≤5个）-> 运到蓝图交付
-> 材料交齐 -> 进入建造倒计时（buildTime）
-> 进度条自下而上填充整个占地 -> 完成（HP恢复）
-> 建造中断（夜晚/饥饿/困倦）进度保存，工程师离开后暂停

【五、居民系统】
--------------------------------
初始：4人，全住大本营，2人在伐木场

AI优先级（从高到低）：
  1. 饥饿 -> 去最近食物仓库/大本营（碰撞体边缘即可）
  2. 困倦 -> 回家（住房/大本营），到家后hidden
  3. 有工作 -> 干活 -> 产出 -> 搬运到对应仓库 -> 循环
  4. 无工作(非工程师) -> 白天巡逻
  工程师(无蓝图) -> 巡逻（橙色）
  工程师(有蓝图) -> GATHERING取材料 -> BUILDING交付 -> CONSTRUCTING施工

居民状态及颜色：
  SLEEPING/GOING_HOME: 蓝灰 #8899cc
  GOING_TO_EAT/EATING: 黄 #ffcc44
  HAULING: 蓝 #44ccff
  WORKING/GOING_TO_WORK: 绿 #44cc88
  GATHERING: 橙 #ff8800
  BUILDING: 深橙 #dd4400
  CONSTRUCTING: 棕橙 #cc6600
  工程师(空闲): 橙 #ff8800
  无业游民: 灰 #aaaaaa
  夜晚无家: 红 #ff4444

居民外观：椭圆身体 10×20px，脚底碰撞体 7px半径(RESIDENT_RADIUS)
任务分配：增至最大工人即满，不可超配
无家可归：夜晚被敌人攻击杀死

【六、工程师系统】
--------------------------------
独立职业(isEngineer=true)，不参与生产建筑工作分配
指派/取消：地图右下角 ➖👷N/M➕ 按钮
  ➕：有空闲且非工程师的居民时显示
  ➖：有工程师时显示
GATHERING: 到最近对应仓库取材料(≤5个) -> BUILDING: 运到蓝图
CONSTRUCTING: 驻扎施工，timer仅工程师在场时走
建造完成 -> 自动找下一蓝图，无则巡逻

【七、仓库与资源】
--------------------------------
资源：食物、木材、石材、铁矿、铁锭
初始自然资源：以大本营为圆心，树木仅在半径 4 格外、石头在 10 格外、铁矿在 20 格外生成；初始可见范围至少有 5 棵树。树簇为 1-10、平均 4；石簇为 1-6、平均 1.5；铁矿以单个为主，低概率为 2-4 块小簇。
每种资源只能放入对应仓库（+大本营兜底）
仓库储存量独立计算，顶部资源栏为全局汇总
仓库进度条：绿<50%，黄50-80%，橙>80%，显示"当前/容量"
大本营可作为所有资源的兜底仓库

isStorage(type) 检测 cat==='storage' 或 type==='town_hall'

【八、昼夜系统】
--------------------------------
一天256s: 白天162s + 黄昏8s + 夜晚78s + 黎明8s
白天: 6:00-22:00 | 夜晚: 22:00-6:00
时钟：地图右上角24h圆形，灰色盘面(白天)黑字，黑色盘面(夜晚)白字

【九、怪物与战斗】
--------------------------------
夜晚从世界边缘刷新，数量=3+天*2，间隔1.5s
寻路目标：大本营
遇建筑攻击摧毁（圆形碰撞检测），天亮撤退
箭塔自动攻击范围内敌人
大本营被摧毁 -> 游戏结束弹窗 -> 可重新开始

【十、世界与相机】
--------------------------------
世界：80×50格，40px/格，3200×2000像素
画布：1200×800显示
相机：左键拖拽空地平移，滚轮缩放(0.3x-3x)，以光标为中心
建筑放置：鼠标中心=建筑中心（像素级居中）

【十一、UI布局】
--------------------------------
网页整体居中，game-container用flex：调试栏 | 地图(canvas) | 建筑栏
  - 调试栏（左侧，120px）：地图外左侧
     速度控制(⏸▶2x5x20x)、无限资源、+👤-👤、☀️🌙、🔓解锁、⬆升本
     🔄重新开始、✏️编辑配置
  - 总资源栏（顶部居中，position:absolute）：地图内中上顶端
     显示食物/木材/石材/铁矿/人口/时间
  - 建筑栏（右侧，120px）：地图外右侧
     所有建筑按钮，未解锁的display:none
  - 工程师栏（position:absolute, bottom:8, right:132px）：地图内右下角
  - 时钟：地图右上角（canvas绘制）
  - 游戏结束弹窗：固定居中

【十二、配置编辑器】
--------------------------------
调试栏 ✏️编辑配置 -> 弹窗
  - 下拉选类别（🏗建筑 / 💰资源）
  - 建筑：选具体建筑 -> 编辑属性(name/icon/hp/maxWorkers/unlock/baseTime/produces/cost) -> ✅应用（即时生效+更新已有建筑HP+更新面板文字）
  - 资源：选具体资源 -> 编辑初始值 -> ✅应用
  - 💾导出JSON / 📂导入JSON
  - 支持 cfgExportJSON() 直接导出

【十三、操作方式】
--------------------------------
左键选建筑 -> 点击地图放置（自动蓝图）
左键已放置建筑 -> 弹出菜单（升级/工人/拆除/移动）
左键拖拽空地 -> 平移相机
右键 -> 取消选择/关闭菜单
滚轮 -> 缩放
点击菜单外 -> 瞬间关闭菜单（mousedown + wheel）

【十四、碰撞系统】
--------------------------------
resolveCollisions(r, nx, ny)：居民移动前碰撞解析
  - 建筑碰撞：跳过道路、跳过仓库/大本营(可进入)、跳过自己工作/居住建筑
  - 居民碰撞：跳过hidden居民，推开+垂直错开避免卡死
  - 世界边界：clamp
findPath(sx,sy,tx,ty,resident)：A*寻路（目前未启用，代码保留）

【十七、疲劳度设计方案（已取消）】
--------------------------------
居民不再拥有疲劳数值、疲劳阈值或疲劳增长配置。日常需求采用 12:00、18:00 固定用餐，作息采用黄昏统一回家与黎明起床；转职限制暂不通过疲劳系统实现。

【十八、两态迷雾】
--------------------------------
- 世界初始完全被迷雾遮挡，仅保留“当前可见”与“完全不可见”，不记录最后一次探索结果。
- 大本营驱散半径 8 格；灯为 1×1 设施，消耗 5 木材、HP 50、建造 2 秒，驱散半径 5 格。
- 其他已经完成的建筑只驱散自身外接圆；蓝图、地板不提供视野。
- 居民提供半径 2 格动态视野，守卫提供半径 3 格动态视野。
- 所有建筑和地板的完整占地必须位于当前可见区域才能放置。迷雾中的敌人不可见，且不能被箭塔攻击。
- 敌人在进入夜晚时根据玩家当前视野轮廓预先选择生成点，必须位于距离清晰区至少 2 格的最近迷雾带；整夜不重新扫描或换点。
- 夜袭总数拆为多波，每波单位在持续时间内随机错峰生成；后期增加波数，不显示波次提示。
- 箭塔攻击使用飞行箭矢，箭矢命中目标后才结算伤害。
- 护林员、矿场、箭塔、灯等拥有作用范围的建筑，在放置蓝图和移动预览时必须显示圆形作用范围；灯的点亮范围为暖黄色圆圈。
- 无业居民与空闲工程师的巡逻范围仅限已完成建筑驱散的稳定视野，不能进入迷雾探索。
- 夜晚大本营与灯提供视觉点光：通过局部减弱夜幕恢复接近白天的原始色彩，并向外缓慢过渡；半径与其迷雾驱散范围一致，不额外驱散迷雾。
- 后续地图关键点首次被探明时可额外显示标记；该功能尚未进入实现范围。
