# 设计变化记录

按时间顺序记录每次设计变更。

---

## 2026-07-11

### 初始策划
- 确立俯视网格塔防 + 殖民模拟方向
- 部落冲突世界观，简约几何体美术

### 居民系统
- 居民饿了去储具取食，困了回家睡觉
- 劳动力全局池，自由分配，≤所需工人
- 无业游民白天巡逻，夜晚回家
- 拆除建筑后工人变无业游民

### 昼夜系统
- 白天 180s / 夜晚 60s → 后调整为白天 162s / 夜晚 78s
- 夜晚 22:00 ~ 6:00，白天 6:00 ~ 22:00
- 新增 24 小时圆形时钟

### 建筑交互
- 左键点击 → 弹出菜单：升级/增减工人/拆除/移动
- 升级非即时，消耗资源+时间
- 增减工人点击不关菜单
- 拆除直接删除，移动半透明虚影

### 人口系统
- 初始 4 人，基础上限 4，住房 +4/座
- 大本营可住人（初始4人，每升一级+4）
- 大本营招募：消耗 5 食物/人，10s
- 招募支持队列（可多次点击）

### 大本营升级解锁系统
- 移除科技树和研究所
- Lv.1: 农场、伐木场、箭塔
- Lv.2: 住房、木材仓库、食物仓库
- Lv.3: 采石场、石材仓库、城墙
- Lv.4: 铁矿、炼铁机、铁矿仓库、铁锭仓库
- 大本营升级消耗：Lv1→2: 20木, Lv2→3: 50木, Lv3→4: 200木+50石
- 建筑等级上限 = 大本营等级
- 升级弹窗显示解锁预览

### 房屋容量系统
- 住房容量 = 4 + (等级-1)×2
- 大本营容量 = 4 + (等级-1)×4
- 满员后 homeless，夜晚红色标记，可被敌人杀死
- 房屋被摧毁后自动找其他有空位的住房

### 初始配置简化
- 初始建筑：大本营 + 伐木场（左一上一格）
- 4 居民全住大本营，2 人在伐木场

### 调试功能
- 速度控制：暂停/1x/2x/5x/20x
- 资源修改、无限资源、增减人口
- 日夜切换、一键解锁、一键升本
- 重新开始

## 2026-07-12

### 碰撞系统重构
- 建筑圆形碰撞体（collisionRadius）
- 居民椭圆身体（10×20px）+ 脚底 7px 碰撞圈
- 建筑虚线占地 + 实心圆
- resolveCollisions 统一碰撞解析
- 居民间碰撞推开 + 垂直错开防死锁
- 仓库/大本营可进入（无碰撞）
- 选中建筑高亮改为金色虚线圆

### 蓝图建造系统（重大变更）
- 放置建筑 → 创建蓝图（50%透明，HP=1），不检查资源
- 工程师从仓库取材料（≤5个/次）→ 运到蓝图交付
- 材料交齐 → 建造倒计时（buildTime）→ 进度条自下而上 → 完成
- 每个蓝图一对一工程师
- 建造按 FIFO 顺序
- 建造中断（夜晚/饥饿/困倦）进度保存

### 工程师系统
- 独立职业（isEngineer），不参与生产工作分配
- 右下角 +/- 按钮指派/取消
- 施工时驻场（CONSTRUCTING），计时器仅在工程师在场时走
- 建造完成自动找下一蓝图

### 建造时间
- Lv.1: 2s, Lv.2: 5s, Lv.3: 8s, Lv.4: 11s（每级+3s）
- 升级时间同建造时间

### 仓库系统
- 新增食物仓库
- 资源分仓存取（food→food_storage, wood→wood_storage, etc.）
- 大本营作为万能仓库兜底
- 仓库进度条：存储量/容量 + 颜色（绿/黄/橙）

### 居民状态颜色区分
- 工程师橙色系（GATHERING/BUILDING/CONSTRUCTING/idle）
- 无业游民灰色
- 工人绿色，搬运蓝色，进食黄色，睡眠蓝灰

### 夜晚行为
- 居民到家后 hidden（消失），黎明 unhidden（出现）
- 工程师夜晚回家也消失

### 配置编辑器
- 游戏内 JSON 编辑器
- 建筑属性编辑（name/icon/hp/maxWorkers/unlock/baseTime/produces/cost）
- 资源初始值编辑
- 应用即时生效，导出/导入 JSON

### 新增/移除建筑
- 新增：食物仓库（🍗）
- 移除：研究所、科技树（改为大本营解锁）

### 全局资源栏
- 工程师栏从总资源栏分离，移到地图右下角
- 导出 JSON 按钮移到配置编辑器内，调试栏只剩编辑配置
