# 设计变化记录

按时间顺序记录每次设计变更。

---

## 2026-07-11

### 初始策划
- 确立俯视网格塔防 + 殖民模拟方向
- 部落冲突世界观，简约几何体美术

### 居民系统
- 居民饿了去储具取食，困了回家睡觉
- 劳动力全局池，自由分配，≤所需工人
- 无业游民白天巡逻，夜晚回家
- 拆除建筑后工人变无业游民

### 昼夜系统
- 白天 180s / 夜晚 60s → 后调整为白天 162s / 夜晚 78s
- 夜晚 22:00 ~ 6:00，白天 6:00 ~ 22:00
- 新增 24 小时圆形时钟

### 建筑交互
- 左键点击 → 弹出菜单：升级/增减工人/拆除/移动
- 升级非即时，消耗资源+时间
- 增减工人点击不关菜单
- 拆除直接删除，移动半透明虚影

### 人口系统
- 初始 4 人，基础上限 4，住房 +4/座
- 大本营可住人（初始4人，每升一级+4）
- 大本营招募：消耗 5 食物/人，10s
- 招募支持队列（可多次点击）

### 大本营升级解锁系统
- 移除科技树和研究所
- Lv.1: 农场、伐木场、箭塔
- Lv.2: 住房、木材仓库、食物仓库
- Lv.3: 采石场、石材仓库、城墙
- Lv.4: 铁矿、炼铁机、铁矿仓库、铁锭仓库
- 大本营升级消耗：Lv1→2: 20木, Lv2→3: 50木, Lv3→4: 200木+50石
- 建筑等级上限 = 大本营等级
- 升级弹窗显示解锁预览

### 房屋容量系统
- 住房容量 = 4 + (等级-1)×2
- 大本营容量 = 4 + (等级-1)×4
- 满员后 homeless，夜晚红色标记，可被敌人杀死
- 房屋被摧毁后自动找其他有空位的住房

### 初始配置简化
- 初始建筑：大本营 + 伐木场（左一上一格）
- 4 居民全住大本营，2 人在伐木场

### 调试功能
- 速度控制：暂停/1x/2x/5x/20x
- 资源修改、无限资源、增减人口
- 日夜切换、一键解锁、一键升本
- 重新开始

## 2026-07-12

### 碰撞系统重构
- 建筑圆形碰撞体（collisionRadius）
- 居民椭圆身体（10×20px）+ 脚底 7px 碰撞圈
- 建筑虚线占地 + 实心圆
- resolveCollisions 统一碰撞解析
- 居民间碰撞推开 + 垂直错开防死锁
- 仓库/大本营可进入（无碰撞）
- 选中建筑高亮改为金色虚线圆

### 蓝图建造系统（重大变更）
- 放置建筑 → 创建蓝图（50%透明，HP=1），不检查资源
- 工程师从仓库取材料（≤5个/次）→ 运到蓝图交付
- 材料交齐 → 建造倒计时（buildTime）→ 进度条自下而上 → 完成
- 每个蓝图一对一工程师
- 建造按 FIFO 顺序
- 建造中断（夜晚/饥饿/困倦）进度保存

### 工程师系统
- 独立职业（isEngineer），不参与生产工作分配
- 右下角 +/- 按钮指派/取消
- 施工时驻场（CONSTRUCTING），计时器仅在工程师在场时走
- 建造完成自动找下一蓝图

### 建造时间
- Lv.1: 2s, Lv.2: 5s, Lv.3: 8s, Lv.4: 11s（每级+3s）
- 升级时间同建造时间

### 仓库系统
- 新增食物仓库
- 资源分仓存取（food→food_storage, wood→wood_storage, etc.）
- 大本营作为万能仓库兜底
- 仓库进度条：存储量/容量 + 颜色（绿/黄/橙）

### 居民状态颜色区分
- 工程师橙色系（GATHERING/BUILDING/CONSTRUCTING/idle）
- 无业游民灰色
- 工人绿色，搬运蓝色，进食黄色，睡眠蓝灰

### 夜晚行为
- 居民到家后 hidden（消失），黎明 unhidden（出现）
- 工程师夜晚回家也消失

### 配置编辑器
- 游戏内 JSON 编辑器
- 建筑属性编辑（name/icon/hp/maxWorkers/unlock/baseTime/produces/cost）
- 资源初始值编辑
- 应用即时生效，导出/导入 JSON

### 新增/移除建筑
- 新增：食物仓库（🍗）
- 移除：研究所、科技树（改为大本营解锁）

### 全局资源栏
- 工程师栏从总资源栏分离，移到地图右下角
- 导出 JSON 按钮移到配置编辑器内，调试栏只剩编辑配置

## 2026-07-13（重大重构）

### 守卫系统重构
- 守卫从"可指派职业"改为独立人口类型，大本营招募（20食物/15s）
- HP=80, 攻击=12/0.6s, 搜索 450px
- 不吃饭不睡觉，白天隐藏，夜晚战斗
- AI: 玩家分配到箭塔的守卫前往操控；未分配守卫机动战斗
- 建筑被攻击时巡逻守卫优先冲过去

### 地图扩至150×150
- 80×50 → 150×150（6000×6000px），大本营居中 (74, 74)

### 伐木场移除
- 木材通过砍真实树木获取，不再有伐木场建筑

### 道路系统移除
- 全部道路/costGrid/流场/Dijkstra 删除，寻路回退到简单直线移动

### 地板系统
- 新增 5×5 地板，1木/块，单击放置，独立 G.floorMask 网格，1.5x移速

### 砍伐系统
- 🪓砍伐/❌取消按钮（右下角），框选标记树木，空闲村民自动砍（3s）

### 地图自然资源
- 树(~150)、石矿(~25)、铁矿(~15) 随机生成

### 蓝图美术+住房
- 蓝图不显示常规 bar，住房蓝图不增加人口上限

### 碰撞
- home/workplace 只在离开时穿透，buildTarget 始终允许接近

## 当前工作批次（经济与表现修正）

- 生产进度改为按建筑统一结算，避免每名工人重复推进同一进度
- 生产效率改为当前已到岗工人，搬运中的工人不计入
- 仓库容量正式生效，全局资源栏改为具体仓库库存的汇总
- 工程师取料、居民吃饭、升级和生产消耗统一从实际仓库扣除
- 采石场和铁矿必须靠近对应资源点，矿点可持续生产
- 炼铁机增加配方：消耗铁矿生产铁锭
- 明确资源采集边界：只有树木可直接砍伐，石矿和铁矿必须通过矿场开采
- 护林员树木归属护林员小屋，普通闲置村民不再抢收自动林场
- 增加视口裁剪、HUD 降频、农场连携缓存和资源点清理
- 更新地面、资源点、建筑结构、状态提示和界面色彩

### 界面视觉重构
- 建筑目录从单列按钮改为三栏 Grid 布局，左右面板固定宽度，地图区域自适应
- 建筑目录按设施、生产、仓储、防御分区
- 移除游戏界面中的 emoji 图标，改用中文单字徽标、Canvas 几何图形和颜色编码；地板、住房和大本营不显示标识
- 施工班组与林务操作整合到地图右下角工具条

### 建筑交互点
- 取消生产建筑固定工作点，工人恢复前往生产建筑中心附近工作。
- 仓库和蓝图交互恢复以建筑中心为目标；箭塔仍使用塔下守卫驻守点。
- 移动避障增加建筑碰撞范围检测和连续坐标绕行点，直线可通行时仍保持直达。
- 石矿和铁矿新增移动碰撞体并纳入绕行检测；树木和树苗保持可穿行。
- 修正返回工作岗位路径只检查部分障碍物的问题，绕行点两段路径均检查全部障碍物。
- 寻路改为事件驱动重算：建筑布局版本变化立即失效旧路径，移动中按 0.25 秒验证并在卡住 0.5 秒后强制重算。
- 批量搬运增加来源追踪、部分入库和居民移除返还，避免产物丢失或生产建筑因搬运工失效锁死。
- 局部绕行支持连续多段规划，可依次绕开多个建筑或矿石障碍。
- 生产建筑新增 5 个产物内部暂存，满批后由一名工人整批搬运；护林员同步支持批量携带木材。
