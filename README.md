# 🏘 殖民模拟塔防

部落冲突风格的俯视网格塔防游戏，**白天生产建设，夜晚抵御怪物**。纯 HTML/CSS/JS 单文件（约 2000+ 行），浏览器直接打开即玩。

## 🎮 快速开始

浏览器打开 `game.html` 即可，无需安装任何依赖。

## 🕹 操作

| 操作 | 方式 |
|------|------|
| 放置建筑 | 右侧面板选择 → 左键地图空白处 |
| 建筑交互 | 左键已放置的建筑 |
| 移动建筑 | 右键菜单 → 移动 → 左键确认位置 |
| 移动视角 | 左键拖拽空地 |
| 缩放 | 滚轮（以光标为中心，0.3x ~ 3x） |
| 取消/关闭 | 右键 |
| 铺设地板 | 选中地板 → 按住左键拖拽（1 木/格） |
| 框选砍树 | 🪓 砍伐模式 → 左键拖拽框选树木 |
| 取消砍树标记 | ❌ 按钮 → 左键拖拽框选取消 |

## 🏗 建筑系统

### 解锁等级

升级大本营逐步解锁新建筑，建筑等级上限 = 大本营等级：

| 大本营等级 | 升级消耗 | 解锁建筑 |
|-----------|---------|---------|
| Lv.1 | — | 农场、箭塔、地板 |
| Lv.2 | 20 木 | 住房、木材仓库、食物仓库 |
| Lv.3 | 50 木 | 采石场、石材仓库、城墙 |
| Lv.4 | 200 木 + 50 石 | 铁矿、炼铁机、铁矿仓库、铁锭仓库 |
| Lv.5 | 500 木 + 200 石 | 自动箭塔 |

### 建筑交互菜单

点击已放置建筑弹出菜单：
- **升级** — 消耗资源提升建筑等级。住房 +2 人口，大本营 +4 人口 + 解锁新建筑
- **增减工人** — 分配/移除生产工人（不可超过建筑上限）
- **移动** — 半透明虚影拖到新位置（大本营不可移动）
- **拆除** — 删除建筑（大本营不可拆除）
- 大本营独有：
  - **招募居民**（5 食物 / 人，10 秒）
  - **招募守卫**（20 食物 / 人，15 秒）

### 蓝图建造流程

放置建筑 → 创建蓝图（半透明，HP=1，不显示常规 UI）→ **工程师**从仓库取材料（每次 ≤5 个）→ 运到蓝图交付 → 材料交齐 → 建造倒计时 → **建筑完成**（HP 恢复，住房人口生效）。

建造时间按解锁等级递增：Lv.1 = 2s，Lv.2 = 5s，Lv.3 = 8s，Lv.4 = 11s，Lv.5 = 14s。

## 👥 居民

### 行为优先级（从高到低）

1. **饥饿** → 去最近食物仓库 / 大本营取食
2. **困倦** → 回家（住房 / 大本营）睡觉（到家隐藏）
3. **有工作** → 去工作场所 → 生产 → 搬运产物到对应仓库 → 循环
4. **空闲** → 有标记树木优先砍树 → 否则白天巡逻，夜晚回家

生产效率 = 分配人数 / 所需工人（上限 100%）。

### 初始配置

初始 4 人，全部住大本营，其中 2 人为工程师。

### 移动方式

流场寻路系统 — 所有居民通过流场在地图上移动，地板可提供 1.5x 移速加成。

### 状态颜色

| 状态 | 颜色 | 说明 |
|------|------|------|
| 睡觉 / 回家 | 🔵 蓝灰 | SLEEPING / GOING_HOME |
| 进食 / 觅食 | 🟡 黄 | EATING / GOING_TO_EAT |
| 搬运 | 🔵 蓝 | HAULING |
| 工作 | 🟢 绿 | WORKING / GOING_TO_WORK |
| 砍树 | 🟢 绿 | GOING_TO_CHOP / CHOPPING |
| 取材料 | 🟠 橙 | GATHERING（工程师） |
| 交付 | 🔴 深橙 | BUILDING（工程师） |
| 施工 | 🟤 棕橙 | CONSTRUCTING（工程师） |
| 工程师空闲 | 🟠 橙 | 巡逻中 |
| 无业游民 | ⚪ 灰 | IDLE / PATROL |
| 夜晚无家 | 🔴 红 | 危险！会被怪物攻击 |

## ⚔️ 守卫系统

守卫是独立兵种，不同于普通居民。白天睡觉，夜晚战斗。

### 属性

| 属性 | 数值 |
|------|------|
| HP | 80 |
| 攻击力 | 12 |
| 攻击 CD | 0.6s |
| 视野 | 450px |
| 招募消耗 | 20 食物 / 人（15 秒） |

### AI 行为

1. **GUARD_FIND_TOWER** → 寻找无人箭塔，前往操纵
2. **GUARD_MANNING** → 站在箭塔上，自动攻击范围内敌人
3. **GUARD_FIGHTING** → 主动搜寻 450px 内敌人交战，优先保护被攻击建筑
4. **GUARD_RETURNING** → 受伤回大本营
5. **GUARD_HEALING** → 治疗 2 游戏小时后满血重返战场

巡逻范围限制在建筑群外围，白天自动回家睡觉（隐藏+回满血）。

> **箭塔需要守卫操纵**才能开火！无人箭塔不会攻击。Lv.5 解锁的自动箭塔则不需要。

## 🌲 地图资源与砍伐

### 自然资源

地图随机生成 ~150 棵树、~25 簇石头（每簇 2-5 个）、~15 簇铁矿（每簇 2-4 个）。

### 砍伐系统

- 点击 🪓 按钮进入砍伐模式 → 左键拖拽框选树木标记
- 空闲非工程师村民自动前往砍树（3 秒/棵）
- 砍倒后获得木材，村民搬运至木材仓库
- ❌ 按钮取消砍伐标记

## 🗄 仓库与资源

五种资源：🍗 食物、🪵 木材、🪨 石材、⚒ 铁矿、🔩 铁锭。

- 每种资源只能放入对应仓库（+ 大本营兜底）
- 仓库独立容量：各 50 单位
- 仓库进度条颜色：绿 < 50% → 黄 50-80% → 橙 > 80%
- 顶部全局资源栏汇总所有仓库存储量

## 🌙 昼夜系统

一天共 256 秒（4 分 16 秒）：

| 时段 | 时长 | 说明 |
|------|------|------|
| ☀️ 白天 | 162s（6:00 ~ 22:00） | 居民生产建设，守卫睡觉 |
| 🌅 黄昏 | 8s | 居民回家，守卫起床，怪物开始刷新 |
| 🌙 夜晚 | 78s（22:00 ~ 6:00） | 怪物进攻，守卫和箭塔防御 |
| 🌄 黎明 | 8s | 怪物撤退，守卫回家睡觉，居民起床 |

右上角圆形 24 小时时钟指示当前时间。

## ⚔️ 怪物与战斗

- 夜晚从地图边缘刷新，数量 = 3 + 天数 × 2，间隔约 1.5s
- 优先攻击守卫 → 攻击建筑 → 直冲大本营
- 箭塔（需守卫操纵）和自动箭塔自动攻击范围内敌人
- 天亮怪物撤退
- 无家可归的居民夜晚会被怪物攻击杀死
- 大本营被摧毁 → 游戏结束

## 🌍 世界

- 150 × 150 格，每格 40px，总 6000 × 6000 像素
- 显示画布自适应窗口大小，支持相机平移和缩放（0.3x ~ 3x）
- 建筑占地：1×1 格（城墙/箭塔/自动箭塔）、2×2 格（大部分建筑）、5×5 格（地板）

## ✏️ 配置编辑器

调试面板 → ✏️ 编辑配置，弹出 JSON 配置编辑器：

- 编辑建筑属性：名称、HP、工人上限、解锁等级、生产时间、产物、造价等
- 编辑资源初始值
- 💾 导出 JSON / 📂 导入 JSON 配置文件
- 修改即时生效（已有建筑 HP 同步更新、面板文字刷新）
