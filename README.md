# 🏘 殖民模拟塔防

部落冲突风格的俯视网格塔防游戏，**白天生产建设，夜晚抵御怪物**。纯 HTML/CSS/JS 单文件（约 1800 行），浏览器直接打开即玩。

## 🎮 快速开始

浏览器打开 `game.html` 即可，无需安装任何依赖。

## 🕹 操作

| 操作 | 方式 |
|------|------|
| 放置建筑 | 右侧面板选择 → 左键地图空白处 |
| 建筑交互 | 左键已放置的建筑 |
| 移动建筑 | 右键菜单 → 移动 → 左键确认位置 |
| 移动视角 | 左键拖拽空地 |
| 缩放 | 滚轮（以光标为中心，0.3x ~ 3x） |
| 取消/关闭 | 右键 |

## 🏗 建筑系统

### 解锁等级

升级大本营逐步解锁新建筑：

| 大本营等级 | 升级消耗 | 解锁建筑 |
|-----------|---------|---------|
| Lv.1 | — | 农场、伐木场、箭塔 |
| Lv.2 | 20 木 | 住房、木材仓库、食物仓库 |
| Lv.3 | 50 木 | 采石场、石材仓库、城墙 |
| Lv.4 | 200 木 + 50 石 | 铁矿、炼铁机、铁矿仓库、铁锭仓库 |

建筑等级上限 = 大本营等级。大本营可住人（初始 4 人，每升一级 +4）。住房每升一级 +2 人口。

### 建筑右键菜单

点击已放置的建筑弹出菜单：
- **升级** — 消耗资源升级建筑等级
- **增减工人** — 分配/移除生产工人（不可超过建筑上限）
- **移动** — 半透明虚影拖到新位置，绿/红色指示是否可放置
- **拆除** — 删除建筑，工人变无业（大本营不可拆除）
- 大本营独有：**招募居民**（消耗 5 食物 / 人，10 秒）

### 蓝图建造流程

放置建筑 → 创建蓝图（半透明，HP=1）→ **工程师** 从仓库取材料 → 运到蓝图交付 → 材料交齐 → 建造倒计时 → 建筑完成。

建造时间按解锁等级递增：Lv.1 = 2s，Lv.2 = 5s，Lv.3 = 8s，Lv.4 = 11s。

## 👥 居民 AI

### 行为优先级（从高到低）

1. **饥饿** → 去最近食物仓库 / 大本营取食
2. **困倦** → 回家（住房 / 大本营）睡觉（到家隐藏）
3. **有工作** → 去工作场所 → 生产 → 搬运产物到对应仓库 → 循环
4. **无工作** → 白天巡逻，夜晚回家

生产效率 = 分配人数 / 所需工人（上限 100%）。

### 状态颜色

| 状态 | 颜色 | 说明 |
|------|------|------|
| 睡觉 / 回家 | 🔵 蓝灰 | SLEEPING / GOING_HOME |
| 进食 / 觅食 | 🟡 黄 | EATING / GOING_TO_EAT |
| 搬运 | 🔵 蓝 | HAULING |
| 工作 | 🟢 绿 | WORKING / GOING_TO_WORK |
| 取材料 | 🟠 橙 | GATHERING（工程师） |
| 交付 | 🔴 深橙 | BUILDING（工程师） |
| 施工 | 🟤 棕橙 | CONSTRUCTING（工程师） |
| 工程师空闲 | 🟠 橙 | 巡逻中 |
| 无业游民 | ⚪ 灰 | IDLE / PATROL |
| 夜晚无家 | 🔴 红 | 危险！会被怪物攻击 |

### 👷 工程师系统

工程师是独立职业，不参与生产工作，专门负责建造蓝图。

- 右下角 👷 按钮指派/取消工程师
- 工作流：取材料 → 运到蓝图 → 交付 → 施工倒计时 → 完成
- 建造中断（夜晚/饥饿/困倦）进度保存
- 每个蓝图同一时间只有一位工程师负责

## 🗄 仓库与资源

四种资源：🍗 食物、🪵 木材、🪨 石材、⚒ 铁矿，另有 🔩 铁锭。

- 每种资源只能放入对应仓库（+ 大本营作为兜底）
- 仓库独立容量：各 50 单位
- 仓库进度条颜色：绿色 < 50% → 黄色 50-80% → 橙色 > 80%
- 顶部全局资源栏为所有仓库汇总

## 🌙 昼夜系统

一天共 256 秒（4 分 16 秒）：

| 时段 | 时长 | 说明 |
|------|------|------|
| ☀️ 白天 | 162s（6:00 ~ 22:00） | 生产建设，无怪物 |
| 🌅 黄昏 | 8s（过渡） | 居民回家，怪物开始刷新 |
| 🌙 夜晚 | 78s（22:00 ~ 6:00） | 怪物进攻，箭塔自动防御 |
| 🌄 黎明 | 8s（过渡） | 怪物撤退，居民起床 |

右上角圆形 24 小时时钟，灰色盘面显示白天，黑色盘面显示夜晚，白针指示当前时间。

## ⚔️ 怪物与战斗

- 夜晚从地图边缘刷新，数量 = 3 + 天数 × 2，间隔约 1.5s
- 寻路直冲大本营，遇建筑则攻击摧毁
- 箭塔自动攻击范围内敌人（射程 150，伤害 15，CD 0.8s）
- 天亮怪物撤退
- 无家可归的居民夜晚会被怪物攻击杀死
- 大本营被摧毁 → 游戏结束，可重新开始

## 🌍 世界

- 80 × 50 格，每格 40px，总 3200 × 2000 像素
- 显示画布 1200 × 800，支持相机平移和缩放
- 建筑占地：1×1 格（围墙/箭塔）或 2×2 格（其余建筑）

## ✏️ 配置编辑器

游戏内置 JSON 配置编辑器（调试面板 → ✏️ 编辑配置）：

- 编辑建筑属性：名称、HP、工人上限、解锁等级、生产时间、产物、造价等
- 编辑资源初始值
- 💾 导出 JSON / 📂 导入 JSON 配置文件
- 修改即时生效（已有建筑 HP 同步更新、面板文字刷新）
