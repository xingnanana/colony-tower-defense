# 🏘 殖民模拟塔防

部落冲突风格的俯视网格塔防游戏。**白天生产建设，夜晚抵御怪物。**

纯 HTML/CSS/JS 单文件，浏览器直接打开即玩。

## 🎮 快速开始

浏览器打开 `game.html` 即可，无需安装。

## 🕹 操作

| 操作 | 方式 |
|------|------|
| 放置建筑 | 右侧面板选择 → 左键地图 |
| 建筑交互 | 左键已放置建筑 |
| 移动视角 | 左键拖拽空地 |
| 缩放 | 滚轮 |
| 取消 | 右键 |

## 🏗 建筑与解锁

升级大本营逐步解锁新建筑：

| 大本营 | 消耗 | 解锁 |
|--------|------|------|
| Lv.1 | - | 农场、伐木场、箭塔 |
| Lv.2 | 20木 | 住房、木材仓库 |
| Lv.3 | 50木 | 采石场、石材仓库、城墙 |
| Lv.4 | 200木+50石 | 铁矿、炼铁机、铁矿仓库、铁锭仓库 |

建筑等级上限 = 大本营等级。大本营可住人（初始 4 人，每升一级 +4）。

## 👥 居民 AI

1. 饥饿 → 去最近储具/大本营取食
2. 困倦 → 回家（住房/大本营）睡觉
3. 有工作 → 干活 → 产出 → 搬运到储具 → 循环
4. 无工作 → 白天巡逻，夜晚回家

生产效率 = 分配人数 / 所需工人（上限 100%）。无家可归者夜晚有危险。

## 🌙 昼夜

- ☀️ 白天 162s（6:00~22:00）：生产建设，无怪物
- 🌙 夜晚 78s（22:00~6:00）：怪物进攻，塔防战斗
- 右上角 24 小时圆形时钟
