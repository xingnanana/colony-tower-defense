# 🏘 殖民模拟塔防

部落冲突风格的俯视网格塔防游戏。**白天生产建设，夜晚抵御怪物。**

纯 HTML/CSS/JS 单文件，浏览器直接打开即玩。

## 🎮 快速开始

```bash
# 浏览器直接打开
open game.html
```

无需安装任何依赖。

## 🕹 操作

| 操作 | 方式 |
|------|------|
| 放置建筑 | 右侧面板选择 → 左键点击地图 |
| 建筑交互 | 左键点击已放置建筑 |
| 移动视角 | 左键拖拽空地 |
| 缩放 | 滚轮 |
| 取消选择 | 右键 |

## 🏗 建筑

| 建筑 | 类型 | 说明 |
|------|------|------|
| 🏛 大本营 | 核心 | 中心建筑，被摧毁则游戏结束，可消耗食物招募居民 |
| 🏠 住房 | 设施 | 提供人口上限，居民回家睡觉，升级增加容量 |
| 🌾 农场 | 生产 | 产出食物，最多2人工作 |
| 🪓 伐木场 | 生产 | 产出木材，最多2人工作 |
| ⛏ 采石场 | 生产 | 产出石料，最多3人工作 |
| ⚒ 铁矿 | 生产 | 产出铁矿，最多4人工作 |
| 📦 仓库 | 储存 | 存放资源，居民搬运目的地 |
| 🏹 箭塔 | 塔防 | 自动攻击范围内敌人 |
| 🧱 城墙 | 防御 | 高血量障碍物，阻挡敌人 |
| 🔬 研究所 | 设施 | 产出科技点，解锁科技树 |

## 👥 居民 AI

1. 饥饿 → 去仓库取食
2. 困倦 → 回家睡觉
3. 有工作 → 干活 → 产出 → 搬运到仓库 → 循环
4. 无工作 → 白天巡逻，夜晚回家

生产效率 = 分配人数 / 所需工人（上限 100%）

## 🌙 昼夜循环

- ☀️ 白天 162s（6:00 ~ 22:00）：生产建设，无怪物
- 🌅 黄昏 8s：居民回家，怪物即将出现
- 🌙 夜晚 78s（22:00 ~ 6:00）：怪物进攻，箭塔防御
- 🌄 黎明 8s：怪物撤退，新一天开始

| 科技 | 消耗 | 效果 |
|------|------|------|
| 农业改良 | 30 | 农场效率 +25% |
| 木工技术 | 30 | 伐木效率 +25% |
| 强化城墙 | 30 | 城墙 HP +100 |
| 石工技术 | 40 | 住房等级上限 +2 |
| 采石技术 | 40 | 采石效率 +25% |
| 高级箭术 | 50 | 箭塔射程 +40 伤害 +10 |
| 采矿技术 | 50 | 铁矿效率 +25% |
| 后勤学 | 50 | 居民移速 +30% |
| 居住改良 | 60 | 人口上限 +2 |
| 要塞化 | 60 | 大本营 HP +200 |

左侧面板提供测试功能：速度控制、无限资源、人口增减、日夜切换、资源编辑。
