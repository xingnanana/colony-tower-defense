========================================
      殖民模拟塔防游戏 - 策划案 v1.0
========================================

【一、游戏概述】
--------------------------------
类型：殖民模拟 + 塔防
视角：俯视网格地图
美术：简约几何体风格
世界观：部落冲突风格（原始部落+奇幻元素，玩家=蓝灰方块，敌人=红橙三角）
核心循环：白天生产建设 → 夜晚塔防战斗 → 循环


【二、核心循环】
--------------------------------
白天阶段：
  居民劳作 → 产出资源 → 搬运至储具 → 消耗资源建造/升级建筑 → 扩张殖民地

夜晚阶段：
  怪物从地图边缘刷新 → 智能寻路冲向殖民地核心
  → 遇到建筑攻击并摧毁 → 防御塔自动攻击敌人 → 天亮怪物撤退

昼夜交替：
  天亮 → 怪物撤退/被消灭 → 居民起床回到岗位
  天黑 → 居民回屋睡觉 → 怪物出现 → 塔防战斗开始


【三、建筑系统】
--------------------------------
所有建筑统一放置在网格上。

建筑分类：

  1. 生产类建筑
     - 产出基础资源（木材/石料/铁矿/食物等）
     - 每座建筑有"所需工人"属性
     - 实际生产时间 = 基础生产时间 ÷（分配人数 ÷ 所需工人）
     - 分配人数不能超过所需工人
     - 效率上限 100%（4人全满即为满效）
     - 举例：铁矿基础10秒产1个，需4人满效，最多分配4人
         4人开采 → 10秒/个
         2人开采 → 20秒/个

  2. 储存类建筑（储具）
     - 存放各类资源
     - 资源必须搬运到储具才算"入库可用"
     - 储具位置影响搬运效率（离生产建筑越远搬运越慢）

  3. 塔防类建筑
     - 攻击来犯敌人
     - 类型可能包括：箭塔、炮台、减速陷阱等
     - 点击塔防建筑显示攻击范围（半透明圆形）

  4. 设施类建筑
     - 提供功能或增益
     - 如：住房（每座+4人口上限，居民回家睡觉）、道路（加速移动）、工坊（解锁升级）等

  5. 大本营（特殊建筑）
     - 游戏开始时位于地图正中央
     - 玩家的核心建筑，被摧毁则游戏结束
     - 不可拆除、不可移动
     - 点击大本营 → "增加人口"按钮 → 消耗食物 → 10秒后产出1个新居民
     - 高血量（500HP），是重点保护目标

建造消耗：
  - 放置/升级建筑需消耗储具中已入库的资源

建筑交互（左键点击建筑弹出操作按钮）：
  - 升级：消耗资源升级，升级需要时间，完成后 HP×1.3，住房额外 +2 人口上限
  - 增加人数 / 减少人数：调整工人分配，不超过上限。满员/空员时按钮灰色不可点击。点击后菜单不消失，可连续操作
  - 拆除：直接删除建筑，原工人变为无业游民。大本营不可拆除
  - 移动：建筑变为跟随鼠标的半透明虚影，占地网格绿色=可放置红色=不可放置，大本营不可移动
  - 升级/移动/拆除点击后菜单关闭；增减工人点击后菜单保持打开


【四、居民系统】
--------------------------------
居民来源：
  - 初始给予一批
  - 大本营招募：点击大本营 → 增加人口，消耗食物，10秒后产出1个居民
  - 住房自动缓慢增长人口

劳动力分配：
  - 全局共享劳动力池
  - 玩家自由将居民分配到各生产建筑
  - 每个建筑分配人数不能超过其"所需工人"上限

居民AI行为（优先级从高到低）：
  1. 饥饿 → 前往储具取食物进食
  2. 困倦 → 回家（住房）睡觉
  3. 有分配工作 → 前往生产建筑干活
     → 产出资源后 → 搬运资源至最近储具 → 返回岗位继续
  4. 无工作 → 白天在地图内巡逻游荡，夜晚仍回家睡觉

居民生死：
  - 夜晚敌人会攻击居民
  - 居民被杀后需等待住房重新增长补充


【五、怪物系统】
--------------------------------
出现时机：夜晚
刷新位置：地图边缘
寻路方式：智能寻路，目标为殖民地核心（或某关键建筑）

行为：
  - 遇建筑 → 攻击并摧毁
  - 会寻找防御薄弱点突破
  - 天亮后剩余怪物撤退

敌我辨识：
  - 玩家方：蓝灰调方块/几何体（部落村庄）
  - 敌人方：红橙调三角/几何体（哥布林、敌对部落、奇幻生物等）

敌人类型（部落冲突风格）：
  - 哥布林：基础杂兵，数量多、血量低
  - 蛮族战士：中等血量伤害
  - 巨人/大型生物：高血量，优先攻击建筑
  - 飞行单位：无视地形/围墙，直冲核心
  - 后续可扩展：法师、飞龙等


【六、昼夜系统】
--------------------------------
一天 = 256 秒（白天162s + 黄昏8s + 夜晚78s + 黎明8s）
白天：早上6:00 ~ 晚上10:00
夜晚：晚上10:00 ~ 早上6:00
时钟：右上角圆形钟表，灰色盘面(白天6:00-22:00)黑字，黑色盘面(夜晚22:00-6:00)白字，纯白指针

白天：
  - 居民活跃（干活、搬运、进食）
  - 玩家可自由建造/分配/调整
  - 无怪物

夜晚：
  - 居民自动回屋睡觉
  - 怪物出现
  - 防御塔自动战斗
  - 玩家可操作有限（或仅观察）


【七、资源系统】
--------------------------------
资源类型：
  - 食物（居民消耗、生存所需）
  - 木材（基础建材）
  - 石料（进阶建材）
  - 铁矿（高级建材/武器）

流转方式：
  生产建筑产出 → 工人搬运 → 储具入库 → 全局可用

关键规则：
  - 未入库的资源不可用于建造
  - 储具容量有限


【八、胜利与失败条件】
--------------------------------
待定（可在后续版本中补充）

胜利方向：
  - 存活指定天数
  - 消灭所有敌人波次
  - 达成人口/建筑目标

失败方向：
  - 大本营被摧毁（立即游戏结束）
  - 居民全部死亡


【九、调试功能】
--------------------------------
  - 速度控制：暂停 / 1x / 2x / 5x 倍速
  - 资源修改：直接修改食物/木材/石料/铁矿数量
  - 调试面板位于游戏地图左侧，与右侧建筑面板对称


【十、后续可扩展内容】
--------------------------------
  - 科技树/大本营等级系统
  - 更多敌人类型（法师、飞龙、亡灵等）
  - 英雄/酋长单位
  - 部落法术（治疗、雷电、狂暴等）
  - 地形/资源点随机生成
  - 天气系统（影响生产效率或怪物行为）
  - 存档/读档
  - 关卡制或沙盒无限模式
