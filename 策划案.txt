========================================
      殖民模拟塔防游戏 - 策划案 v2.0
========================================

【一、游戏概述】
--------------------------------
类型：殖民模拟 + 塔防
视角：俯视网格地图，80×50 格世界，支持相机平移缩放
美术：简约几何体风格
世界观：部落冲突风格（原始部落+奇幻元素，玩家=蓝灰方块，敌人=红橙三角）
核心循环：白天生产建设 → 夜晚塔防战斗 → 循环


【二、核心循环】
--------------------------------
白天阶段：
  居民劳作 → 产出资源 → 搬运至储具/大本营 → 消耗资源建造/升级建筑 → 扩张殖民地

夜晚阶段：
  怪物从地图边缘刷新 → 智能寻路冲向大本营
  → 遇到建筑攻击并摧毁 → 防御塔自动攻击敌人 → 天亮怪物撤退

昼夜交替：
  天亮 → 怪物撤退/被消灭 → 居民起床回到岗位
  天黑 → 居民回屋睡觉 → 怪物出现 → 塔防战斗开始


【三、建筑系统】
--------------------------------
所有建筑统一放置在网格上。升级大本营解锁新建筑。

大本营升级消耗：
  - Lv.1 → 2：20 木材
  - Lv.2 → 3：50 木材
  - Lv.3 → 4：200 木材 + 50 石材

建筑解锁等级：
  一本（Lv.1）：农场、伐木场、箭塔
  二本（Lv.2）：住房、木材仓库
  三本（Lv.3）：采石场、石材仓库、城墙
  四本（Lv.4）：铁矿、炼铁机、铁矿仓库、铁锭仓库

建筑等级上限 = 大本营等级（大本营本身无上限）

建筑分类：

  1. 生产类建筑
     - 农场（Lv.1，2人，食物）、伐木场（Lv.1，2人，木材）
     - 采石场（Lv.3，3人，石材）、铁矿（Lv.4，4人，铁）
     - 炼铁机（Lv.4，2人，铁锭）
     - 实际生产时间 = 基础生产时间 ÷（分配人数 ÷ 所需工人）
     - 分配人数不能超过所需工人

  2. 储存类建筑
     - 木材仓库（Lv.2）、石材仓库（Lv.3）
     - 铁矿仓库（Lv.4）、铁锭仓库（Lv.4）
     - 大本营也可存放资源
     - 资源必须搬运入库才算可用

  3. 塔防类建筑
     - 箭塔（Lv.1）：自动攻击范围内敌人，点击显示攻击范围
     - 城墙（Lv.3）：高血量障碍物

  4. 设施类建筑
     - 住房（Lv.2）：+4 人口上限/座，升级 +2
     - 大本营：核心建筑，被摧毁则游戏结束，可住人（初始4人，每升一级+4）

建筑交互（左键点击建筑弹出操作按钮）：
  - 升级：消耗资源，升级需要时间，完成后 HP×1.3
  - 增加/减少工人：不超过上限，满员/空员时按钮灰色。点击不关菜单
  - 拆除：删除建筑，工人变无业游民。大本营不可拆除
  - 移动：半透明虚影跟随鼠标，绿色可放置/红色不可放置。大本营不可移动
  - 升级/移动/拆除关闭菜单；增减工人保持菜单


【四、居民系统】
--------------------------------
居民来源：
  - 初始 4 人，全部居住在大本营
  - 大本营招募：消耗食物，10s/人，可排队多次
  - 住房自动缓慢增长

劳动力分配：
  - 全局共享劳动力池，玩家自由分配
  - 每个建筑不能超过所需工人上限

居民AI行为（优先级从高到低）：
  1. 饥饿 → 到最近储具/大本营取食
  2. 困倦 → 回家（住房/大本营）睡觉
  3. 有工作 → 干活 → 产出 → 搬运到最近储具/大本营 → 循环
  4. 无工作 → 白天巡逻，夜晚回家

房屋容量：
  - 住房：4 + (等级-1)×2
  - 大本营：4 + (等级-1)×4
  - 满员时居民 homeless，夜晚红色标记，会被敌人攻击


【五、怪物系统】
--------------------------------
出现时机：夜晚
刷新位置：世界边缘
目标：大本营

行为：
  - 遇建筑 → 攻击并摧毁
  - 天亮后剩余怪物撤退

敌人类型（部落冲突风格）：
  - 哥布林、蛮族战士、巨人、飞行单位等


【六、昼夜系统】
--------------------------------
一天 = 256 秒（白天162s + 黄昏8s + 夜晚78s + 黎明8s）
白天：早上6:00 ~ 晚上10:00 / 夜晚：晚上10:00 ~ 早上6:00
时钟：右上角圆形，灰色盘面(白天)黑字，黑色盘面(夜晚)白字，纯白指针


【七、资源系统】
--------------------------------
资源类型：食物、木材、石材、铁矿、铁锭
流转方式：生产建筑产出 → 工人搬运 → 储具/大本营入库 → 全局可用
关键规则：未入库资源不可用于建造，储具容量有限


【八、胜利与失败条件】
--------------------------------
胜利方向：存活指定天数 / 消灭所有敌人波次 / 达成人口或建筑目标
失败方向：大本营被摧毁（立即游戏结束）/ 居民全部死亡


【九、调试功能】
--------------------------------
  - 速度控制：暂停 / 1x / 2x / 5x / 20x
  - 无限资源、增减人口、一键解锁、一键升本
  - 日夜切换、资源直接修改
  - 面板位于地图左侧


【十、后续可扩展内容】
--------------------------------
  - 更多敌人类型、英雄/酋长单位
  - 部落法术（治疗、雷电、狂暴等）
  - 地形/资源点随机生成
  - 天气系统、存档/读档
